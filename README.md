# 📊Power-Query-Data-Cleaning

## 👩🏽‍💻 Project Overview

This project demonstrates my data cleaning and transformation skills using **Power Query in Microsoft Excel**. The goal was to take a messy sales dataset and transform it into a clean, analysis ready format.

As a data analyst with a healthcare background, I am building strong data preparation skills essential for real world analytics workflows.

---

## 🎯 Objectives

- Clean and standardize customer names  
- Format Nigerian phone numbers to international format (+234)  
- Validate and correct email addresses  
- Standardize date fields  
- Detect price outliers  
- Prepare dataset for analysis  

---

## 🧰 Tools Used

- Microsoft Excel  
- Power Query  
- Basic data validation techniques  

---

## 🔧 Data Cleaning Steps

The following transformations were performed in Power Query:

1. Split **Full Name** into First Name and Last Name  
2. Standardized phone numbers to +234 format  
3. Cleaned and validated email addresses  
4. Standardized date formats  
5. Created Month Name column  
6. Flagged price outliers  
7. Verified total sales calculations  
8. Removed inconsistencies in text fields  

---

## 📁 Dataset Description

The dataset contains:

- Customer information  
- Purchase details  
- Product data  
- Sales metrics  
- Contact information  

After cleaning, the data is ready for dashboarding and analysis.
